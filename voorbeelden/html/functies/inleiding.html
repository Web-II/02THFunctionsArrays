Functies

Inleiding

Heel vaak moeten we op veel plaatsen in het script een vergelijkbare actie uitvoeren.
We moeten bijvoorbeeld een mooi bericht weergeven wanneer een bezoeker inlogt, uitlogt en misschien ergens anders.
Functies zijn de belangrijkste "bouwstenen" van het programma. Ze zorgen ervoor dat de code vele malen kan worden gebruikt zonder letterlijk de code te herhalen.

Enkele voordelen van het gebruik van functies:

  • Efficiëntere code
  • Herbruikbaar
  • Elimineert dubbele code
  • Grotere problemen oplossen door kleinere problemen op te lossen
  • Grotere problemen opdelen in kleinere eenheden
  • Er bestaan built-in functies zoals alert(...), Math.round(...)

Een functie declaratie ziet er als volgt uit 

    function functionname(par1, par2, ..., parx) {
        statements
    };
Eerst is er het sleutelwoord function, dan volgt de naam van de functie, gevolgd door een lijst van parameters tussen (). Deze lijst mag eventueel leeg zijn. Tenslotte is er de code van de functie (de function body), tussen {}.
De functie wordt uitgevoerd, door de functie aan te roepen 
    functionname(arg1, arg2, ..., argx);
Een functie kan een waarde retourneren. 

function getAvatar(points) {
    let avatar;
    if (points < 100) {
        avatar = 'Mouse';
    } else if (points < 1000) {
        avatar = 'Cat';
    } else {
        avatar = 'Gorilla';
    }
    return avatar;
}
const myAvatar = getAvatar(335); 
console.log(`my avatar : ${myAvatar}`); //Cat

Oefening

  • Schrijf een functie die n! berekent, waarbij n meegegeven wordt als parameter.
    Bijvoorbeeld 5! = 5 * 4 * 3 * 2 * 1 = 120.
    Roep deze functie op met verschillende parameters 



function faculteit(n) {
    let result = 1;
    for (let i = 1; i <= n; i++) {
        result *= i;
    }
    return result;
}
console.log(faculteit(5));

Oefening

  • v-perfecte getallen zijn strikt positieve gehele getallen waarvan de som van de positieve delers gelijk is aan een veelvoud v van het getal zelf. Een voorbeeld van een 3-perfect getal is 120. De som van alle delers van 120, namelijk 1+2+3+4+5+6+8+10+12+15+20+24+30+40+60+120=360 is immers gelijk aan 3 maal 120
    Schrijf een functie die retourneert of een getal een v-perfect getal is of niet.
    Als het een v-perfect getal is, wordt de waarde van v geretourneerd, anders false 



function vperfect(getal) {
    let somDelers = 0;
    for (let i = 1; i <= getal; i++) {
        if ((getal % i) === 0) somDelers += i;
    }
    return (somDelers % getal) === 0 ? somDelers / getal : false;
}

console.log(vperfect(120));     // 3
console.log(vperfect(121));     // false
