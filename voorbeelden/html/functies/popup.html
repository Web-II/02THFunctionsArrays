JavaScript Popup boxes

De functies alert(message), confirm(question) en prompt(message, default) zijn voorbeelden van built-in functions

  • Alert box 
    alert("I am an alert box");
  • Confirm box 
    const result = confirm("Press a button");
if (result) {
    alert("You pressed OK");
}
else {
    alert("You pressed cancel");
}
  • Prompt box. Als geen defaultwaarde voorzien wordt, wordt null geretourneerd. 
    const name = prompt("Please, enter your name","Harry Potter");
if (name){
    alert(`Hello ${name}! How are you today?`);
}
