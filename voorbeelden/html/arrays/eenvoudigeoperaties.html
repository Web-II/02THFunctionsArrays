Arrays

Operaties

Enkele eenvoudige operaties op een array:

  • Een lege array 
            let pizzas = [];
    OF 
            let pizzas = new Array();
        
    let pizzas = [];
  • Arrays zijn dynamisch (je dient het aantal elementen in de array niet op voorhand te definiëren). Een array kan groeien en krimpen!
  • Je dient ook het type van de elementen in de array niet op te geven. De elementen mogen van een verschillende type zijn.
  • Waarde toekennen (mogen verschillende types zijn) aan een array. 
    let pizzas = [];
pizzas[0] = 'Margherita';
pizzas[1] = 'Mushroom'; 
pizzas[2] = 'Spinach & Rocket';
// Je kan dit ook eenvoudig wijzigen
pizzas[0] = 'Ham & Pineapple';
  • Aanmaken van een array via array literal 
    let pizzas = ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];
let mixedArray = [null, 1, 'two', true, undefined ];
  • Opvragen van 1 element aan de hand van de index
    • Index start van 0. Gebruik []
    • length : aantal elementen in array
    const pizzas = ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];
console.log(pizzas[2]); //Spinach & Rocket
console.log(`eerste pizza ${pizzas[0]}`); //Margherita
console.log(`laatste pizza ${pizzas[pizzas.length-1]}`); //Pineapple & Sweetcorn
  • Je kan ook de volledige array weergeven 
    const pizzas = ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];        
console.log(pizzas); // Array(4) ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];
  • Verwijderen van een element in de array
    delete verwijdert de waarde op deze positie, maar de ruimte bestaat nog steeds en bevat dan de waarde undefined. 
    const pizzas = ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];     
delete pizzas[2];   
console.log(pizzas); // Array(4) ['Margherita', 'Mushroom', undefined, 'Pineapple & Sweetcorn'];
  • Overlopen van een array
    • for lus 
      const pizzas = ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];     
for (let i = 0; i < pizzas.length; i++) {
    console.log(pizzas[i]);
}
    • for - of lus 
      const pizzas = ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];     
for (let value of pizzas) {
    console.log(value);
}
    • Merk op 
      const pizzas = ['Margherita', 'Mushroom', 'Spinach & Rocket', 'Pineapple & Sweetcorn'];
pizzas[30] = 'Vegetarian';
console.log(pizzas.length); // 31     
console.log(pizzas[20]); // undefined
